# Resources

**This file is the main reference list for the project.** All external documentation links live here. Chapter files refer to resources by name (e.g., "See the MDN State machine entry in resources") rather than repeating URLs. When a link changes, update it once here.

---

The Github Repo with the source of truth documents for this project. https://github.com/thetecnoagrarian/Learn-to-Code-Slowly

These are resources for technologies I want to learn and build with.

- [MDN (Mozilla Developer Network)](https://developer.mozilla.org/en-US/) — Web standards, APIs, and developer documentation
- [MDN: State machine](https://developer.mozilla.org/en-US/docs/Glossary/State_machine) — Glossary entry defining state machines
- [MDN: HTTP](https://developer.mozilla.org/en-US/docs/Web/HTTP) — HTTP protocol overview and reference
- [Python 3 Documentation](https://docs.python.org/3/) — Official Python 3 language and standard library docs
- [The Linux Documentation Project (TLDP)](https://tldp.org/) — Free, collaborative Linux documentation
- [Linux Kernel Documentation](https://docs.kernel.org/) — Official Linux kernel docs
- [MQTT](https://mqtt.org/) — Message Queuing Telemetry Transport protocol for IoT messaging
- [SQLite Documentation](https://www.sqlite.org/docs.html) — SQLite database documentation
- [MySQL Documentation](https://dev.mysql.com/doc/) — MySQL relational database documentation
- [Docker Networking](https://docs.docker.com/engine/network/) — Docker container networking documentation
- [Docker Buildx](https://docs.docker.com/build/buildx/) — Docker CLI plugin for multi-platform image builds
- [Express.js](https://expressjs.com/) — Fast, minimal web application framework for Node.js
- [Linode](https://www.linode.com/docs/) — Cloud hosting provider; Linux VMs and infrastructure
- [Shoelace](https://shoelace.style/) — Forward-thinking library of web components built with Lit
- [better-sqlite3](https://github.com/WiseLibs/better-sqlite3) — Synchronous Node.js binding for SQLite3
- [MicroPython](https://docs.micropython.org/) — Python 3 implementation for microcontrollers
- [ESP32 (Espressif)](https://docs.espressif.com/) — ESP32 series microcontrollers; Wi-Fi and Bluetooth
- [ESPHome](https://esphome.io/) — Control ESP8266/ESP32 microcontrollers with YAML configuration
- [Home Assistant](https://www.home-assistant.io/) — Open-source home automation platform; local control and privacy
